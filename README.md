This is a fork of "Interfisch"'s brilliant work for creating an ESP32 based "Annax" flipdot module controller board, software and more.

The goals of this fork are
- Re-layouting the controller board to fit the original ADtranz controller board
- Modifying the ESP32 software to support the original ADtranz flipdot module configuration (2x 25x16, 2x 20x16, 1x 25x16)
- Added another useful test pattern (checkers)
- Made flipdot timing parameters settable through MQTT, to help determine ideal settings
